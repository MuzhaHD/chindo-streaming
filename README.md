# CHINDO - Platform Streaming Donghua

Platform streaming donghua 3D dengan fitur lengkap untuk menonton, berinteraksi, dan mengelola konten.

## 🚀 Fitur

- **Streaming Video**: Multi-server support dengan source switcher
- **Autentikasi**: Login dengan Email/Password dan Google OAuth
- **Watchlist**: Simpan film favorit untuk ditonton nanti
- **Watch Progress**: Lanjutkan menonton dari posisi terakhir
- **Komentar**: Sistem komentar bersarang dengan like
- **Search**: Pencarian film berdasarkan judul, genre, deskripsi
- **Admin Panel**: Kelola film, episode, dan komentar
- **Responsif**: Tampilan optimal di desktop dan mobile

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Database**: MongoDB + Mongoose
- **Auth**: NextAuth v5 (Auth.js)
- **Styling**: Tailwind CSS
- **Language**: TypeScript
